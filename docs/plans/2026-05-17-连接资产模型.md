# 连接资产模型 Implementation Plan

**Goal:** 废弃 `connections.json`，以 `dsl/connections/*.connection.yaml` 作为连接工程资产真值源，并把密钥、本机覆盖和运行态状态拆到 Store / Runtime。

**Architecture:** 新增强类型 `ConnectionSpec` / `ConnectionProtocol`，Tauri 壳层负责从连接资产、环境覆盖和 Store secret/local override 解析出运行时连接定义。`ConnectionManager` 继续只负责运行时治理与 RAII 借出，不读取文件、不解析密钥、不保存工程语义。

**Tech Stack:** Rust / Tauri / React / TypeScript / Vitest / Playwright / cargo-deny。

> **Status:** 已实施（2026-05-17）。ADR-0025 已更新为“已实施”，legacy `connections.json` IPC 与部署时 `runtimeConnections` 已移除。
